Vision for Features

Dashboard

Dashboard for curators to manage experiences and in-country customer service. A dashboard open up the possibility for flash events/sales based on data and gives curators the ability to know how many active app users are in-country.

API Driven Business

Develop business applications behind an API. Open API to developers and businesses so Tastemakers Africa experiences can be booked by third parties. For example someone booking a room on Hotels.ng or jumia.com can add-on an insider experience.

Social Features for App

Introducing social features on the app will make for a better user experience. Messaging, trip planning with friends, and installment payments are some of the things I'd like to build.
